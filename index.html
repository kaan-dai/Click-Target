Click Target Game

How to Play

Click targets as they appear on screen to score points!

You have 2 lives, so be careful!


  • Regular targets -> +1 point

    • BUFFS


  • Green Target -> Extra Life +1
  • Blue Target -> Slow Targets, Time Freeze, 5 sec
  • Purple Target -> Target Size 1.5x, Score Multiplier 2x, 5 sec
  • Yellow Target -> Score Multiplier 2x
  • Rainbow Giant Target -> +5 points

    • DEBUFFS


  • Red Bomb Target -> Lose Life -1
